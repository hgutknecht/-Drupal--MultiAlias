
Description
===============
Multialias module provides an interface for automatically generating multiple aliases per node type based off of patterns defined by the user. Multialias extends the paths set by pathauto.

For more information on Multialias see ...drupal.org (replace with sandbox url)

Requirements
===============
Multialias requires the pathauto module.
